Datenschutzerklärung

1. Verantwortlicher

Betreiber des Discord Bots Proxa Cloud.


2. Welche Daten werden gespeichert?

Zur Bereitstellung der Bot-Funktionen werden ausschließlich technische Informationen gespeichert:


  • Server-ID
  • Kanal-ID
  • Rollen-ID
  • Nachrichten-ID

Nicht gespeichert werden:

  • Nachrichteninhalte
  • Benutzernamen
  • Passwörter
  • IP-Adressen
  • Persönliche Daten

3. Zweck der Speicherung

Die gespeicherten Daten werden ausschließlich benötigt, um Funktionen wie Verify-System, Welcome-System, Stats-System und weitere Servereinstellungen bereitstellen zu können.


4. Weitergabe der Daten

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte.


5. Speicherort

Alle Einstellungen werden lokal auf dem Server gespeichert, auf dem Proxa Cloud betrieben wird.


6. Speicherdauer

Die Daten bleiben gespeichert, bis der Bot vom Server entfernt oder die Einstellungen zurückgesetzt werden.


7. Rechte

Serverbesitzer können jederzeit die Löschung ihrer gespeicherten Daten beantragen.


8. Kontakt

Bei Fragen zum Datenschutz kontaktiere uns über unseren Discord-Server.
